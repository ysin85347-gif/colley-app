COLLEY 제품 렌더 이미지 저장 위치
=====================================

아래 파일명으로 실제 제품 이미지를 이 폴더에 저장하면
앱의 제품 비주얼(홈 장치목록 / 차량관리 진단·상세 / 더보기 제품정보)에
자동으로 표시됩니다. 파일이 없으면 SVG 실루엣이 대신 표시됩니다.

  colley-render.png        기본 이미지 (도킹/대기/충전 상태)
                           - 권장: 3/4 정면 서 있는 컷 (첫 번째 이미지)
  colley-drone-flight.png  드론 비행/이륙/복귀 상태에서 표시
                           - 권장: 드론이 분리/부각된 컷
  colley-drone-docked.png  (선택) 도킹 클로즈업

홈 히어로(실행 장면)
  colley-action.jpg        홈 상단 배너에 표시되는 "COLLEY 실행" 사진
                           - 권장: 로봇이 양 무리를 모는 가로형 컷 (드론 포함)
                           - 없으면 목초지 SVG 장면이 자동 표시됩니다.

권장 사양
  - 배경: 흰색 또는 투명(PNG)
  - 가로 1600px 내외, 정사각형에 가까운 여백 확보
  - 파일 크기 최적화(500KB 이하 권장)

교체 후 브라우저 새로고침하면 즉시 반영됩니다.
